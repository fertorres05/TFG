# TFG – Aplicación de Gestión de Reservas de Vuelos

**Ciclo Formativo de Grado Superior (CFGS) – Desarrollo de Aplicaciones Multiplataforma (DAM)**  

Aplicación Android desarrollada en **Kotlin** destinada a la gestión y el registro de reservas de vuelos aéreos, permitiendo al usuario llevar un control completo de sus viajes.

---

## Descripción
La aplicación permite al usuario:

- Registrar todos sus vuelos.
- Guardar información sobre los **equipajes**, el **número de pasajeros** y el **gasto total** de cada vuelo.
- Mantener un historial organizado de todas sus reservas.
- Calcular automáticamente el **gasto total acumulado** de los vuelos registrados.
- Obtener información actualizada de vuelos mediante una **API REST**.
- Acceder a la aplicación de forma segura mediante **Firebase Authentication**.

El objetivo es ofrecer una herramienta sencilla y completa para la organización de viajes personales.

---

## Características principales
- **Registro de vuelos** con fecha, origen, destino y aerolínea.
- **Gestión de equipajes** por vuelo.
- **Control del número de pasajeros** por reserva.
- **Cálculo del gasto total** de cada vuelo y del conjunto de reservas.
- **Integración con API REST** para la obtención de datos de vuelos.
- **Inicio de sesión seguro** mediante Firebase Authentication.
- **Interfaz intuitiva**, adaptada a dispositivos Android.

---

## Tecnologías utilizadas
- **Lenguaje:** Kotlin + Jetpack Compose  
- **Base de datos:** PostgreSQL  
- **API REST:** Obtención de datos de vuelos  
- **Autenticación:** Firebase Authentication  
- **Control de versiones:** Git y GitHub
